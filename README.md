PlainGraph
==========

A tiny and lightweight .Net Chart Control. It shows 7 types of chart with one data source.
The following types of chart are available:

 - Line
 - LinePoint
 - LineArea
 - Column
 - Stacked Column
 - Stacked Percent Column
 - Pie
